# Sistema de tickets para la unidad #3

##Descripcion
Esto es una miniaplicacion para crear y gestionar una pagina web de tickets de soporte tecnico
Permite a los usuarios enviarSe puede filtrar por prioridad, buscar tickets por texto, marcar como leído/no leído y eliminar tickets.  
El ticket de alta prioridad se resalta y se mantiene arriba en la lista.  tickets con su nombre, correo, su mensaje ademas de su prioridad, tanto puede ser baja como alta
